Mathematical Discussion

The Newton-Raphson Method is a numerical method used to find the approximate root of an equation.

It starts with an initial guess and repeatedly improves the value using the derivative of the function.

\[ x_{n+1}=x_n-\frac{f(x_n)}{f'(x_n)} \]

Where:

  • \(x_n\) = current approximation
  • \(f(x_n)\) = function value
  • \(f'(x_n)\) = derivative value
  • \(x_{n+1}\) = next approximation

Worked Example 1

Find the root of:

\[ f(x)=x^2-4 \]

Derivative:

\[ f'(x)=2x \]

Initial guess:

\[ x_0=3 \]
Iteration xₙ f(xₙ) f'(xₙ) xₙ₊₁ Error
1 3.0000 5.0000 6.0000 2.1667 0.8333
2 2.1667 0.6944 4.3334 2.0064 0.1603
3 2.0064 0.0256 4.0128 2.0000 0.0064

Therefore, the approximate root is \(x = 2.0000\).

Worked Example 2

Find the root of:

\[ f(x)=x^3-x-2 \]

Derivative:

\[ f'(x)=3x^2-1 \]

Initial guess:

\[ x_0=1.5 \]
Iteration xₙ f(xₙ) f'(xₙ) xₙ₊₁ Error
1 1.5000 -0.1250 5.7500 1.5217 0.0217
2 1.5217 0.0021 5.9471 1.5214 0.0004
3 1.5214 0.0000 5.9438 1.5214 0.0000

Therefore, the approximate root is \(x = 1.5214\).
